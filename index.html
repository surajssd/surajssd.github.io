Testing 'fedora' and 'CentOS' kompose package

Posted on March 14, 2017 | written by

I generally do kompose package testing for fedora and CentOS. So here are the steps I follow. Note: All the commands that start with # are inside the container. Fedora Spin the fedora test environment, let’s say fedora 25, I do it in containers. $ docker run -it fedora:25 bash Once inside the container download the latest package from the testing repo: # dnf --enablerepo updates-testing -y install kompose Note: Make sure you download the latest release, there might be conditions that you download the old package.
