Bangalore Kubernetes Meetup April 2017

April 8, 2017 Suraj Deshmukh

2 minute read

  • Twitter

  • Google+

  • Facebook

  • Reddit

  • LinkedIn

  • StumbleUpon

  • Email

Bangalore Kubernetes Meetup was held in Red Hat Bangalore office on April 8th.

The meetup started with talk from Suraj Deshmukh about Kubernetes on CRI-O. Where he explained about the jargons like OCI), CRI, etc. What is CRI-O? How is the architecture of CRI-O? How it is solving certain problem. And then he demoed the use of cri-o with kubernetes. Link to slides.

Then was talk from Dipak Pawar about System and integration testing for k8s/Openshift with arquillian-cube and JBoss Forge. He explained how arquillian-cube can be used with Kubernetes and OpenShift for doing tests of your micro-services deployed on OpenShift/K8s.

After a tea break next talk was by Budhram Gurung on Running OpenShift locally using Minishift. Where he explained what all components is Minishift made of. He then demoed the usage of Minishift.

Last planned talk was by Suraj Narwade on running Docker on ARM/Raspberry PI. He discussed all the problems he faced when trying to get containers run on Raspberry PI. Also his discovery that the container images built on x86_64 won’t run on arm, etc. Link to slides.

Finally a lightening talk which turned into discussion of 30mins, was about Persistent Storage with Kubernetes by Raghavendra Talur. He demoed the gluster feature of doing hyper-converged storage and dynamic provisioning with gluster. He got lot of questions about setting up storage with Kubernetes.

The meetup ended with discussion over snacks. Next meetup was also planned which is gonna be a Kubernetes and OpenShift 101 workshop.

Thanks to Speakers for presenting, Baiju for recording all the sessions, Red Hat for the venue.
